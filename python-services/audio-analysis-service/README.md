# 🎵 Audio Analysis Microservice

Real-time audio analysis service using librosa for speech evaluation.

## Features
- RMS Energy calculation
- Pitch detection (YIN algorithm)
- Tempo/speech rate estimation
- Silence detection and ratio calculation
- Confidence scoring
